## Conventions

- Reference screenshots stay in one canonical folder so `@design-reference/...` works from any prompt.
- For best first-build quality, use `start-here/STAGED_WORKFLOW.md` and update `start-here/SESSION_STATE.md` before every checkpoint or fresh AI session.
- Slash commands are at `.claude/commands/` so Claude Code auto-discovers them.
- All other docs live under their topic folder. When a prompt or rule cites a path, it uses the form `product/PRD.md` or `mobile/MOBILE_APP_ROADMAP.md` вЂ” never bare filenames.
