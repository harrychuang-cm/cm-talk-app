# Phase 1 — design-spec WebView App

This is the fastest synchronized app target. It wraps the generated web product in an installable Expo app.

## Run

```bash
cd mobile/apps/mobile-webview
npm install
EXPO_PUBLIC_WEB_APP_URL=http://127.0.0.1:5174 npm run start
```

Use this for demos, QA, internal review, and early distribution while the native scaffold matures.
