Two kinds of files live here:

- `uiux-flow.json` — screen-level flow auto-generated by `/prd` and `/change`. Renders in Storybook **Product Overview** via `@xyflow/react` + `@dagrejs/dagre`. Schema: `product/templates/UIUX_FLOW.schema.json`. Do NOT hand-author positions.
- `*.md` flow specs — only for complex multi-step interactions (KYC, checkout, recovery). Use `product/templates/FLOW_SPEC_TEMPLATE.md` as the starting point.
